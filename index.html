About

A fully featured tower defense game built with <canvas>. The game is developed for study purposes and is to serve as a teamwork project for Telerik Academy's JavaScript UI and DOM course.

Instructions

Build and upgrade turrets (which costs money and increases score) to kill creeps (which generates money and increases score). Experiment with the turrets; each has a special ability when fully upgraded.
